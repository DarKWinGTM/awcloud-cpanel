# 📋 SUBSCRIPTION INFO Modal - คู่มือการใช้งาน

{% hint style="success" %}

## 📖 ภาพรวม Modal

### 🎯 **การเข้าถึง**
**เข้าถึง:** เมนูระบบ → modal subscriptionInfo

### 🧭 **โครงสร้าง Modal**
มี **2 Tabs** หลัก:

1. **RENEW** - ต่ออายุบริการ
2. **ADDITIONAL** - เพิ่มบริการเสริม

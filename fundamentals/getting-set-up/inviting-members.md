# 🧑 การจัดการผู้ใช้และทีมงาน

## ระบบสิทธิ์ผู้ใช้ AWCLOUD CPanel

### 👑 **Administrator (ผู้ดูแลระบบ)**
- สิทธิ์เต็มในการจัดการระบบทั้งหมด
- เพิ่ม/ลบ/แก้ไข accounts และ configurations
- ตั้งค่าระบบ security และ proxies
- ดูและจัดการ logs ทั้งหมด
- สิทธิ์ในการสำรองและกู้คืนข้อมูล

### ✏️ **Editor (ผู้แก้ไข)**
- จัดการ mining operations และ strategies
- แก้ไข account settings และ configurations
- ดู mining statistics และ reports
- ไม่สามารถเปลี่ยนแปลงการตั้งค่าระบบหลัก

### 👀 **Viewer (ผู้ดู)**
- ดู mining dashboard และ statistics
- เข้าถึง reports และ mining history
- ไม่สามารถแก้ไขการตั้งค่าใดๆ
- เหมาะสำหรับการติดตามผลงาน

### 🎫 **Guest (ผู้เยี่ยมชม)**
- เข้าถึงข้อมูลเฉพาะที่ได้รับอนุญาต
- ดูสถิติเบื้องต้นเท่านั้น
- ไม่สามารถเข้าถึงข้อมูลที่เป็นความลับ

## การเพิ่มสมาชิกใหม่

### ขั้นตอนการเพิ่มผู้ใช้
1. เข้าสู่ระบบด้วยสิทธิ์ Administrator
2. ไปที่เมนู "User Management"
3. คลิก "Add New User"
4. กรอกข้อมูลผู้ใช้และเลือกระดับสิทธิ์
5. ส่งข้อมูล login ให้ผู้ใช้ใหม่

### การจัดการ Team Mining
- ตั้งค่า team-based mining strategies
- กระจาย workload ระหว่างสมาชิก
- ระบบ reporting แยกตาม user
- การแบ่งปันผลตอบแทนตามสัดส่วน
